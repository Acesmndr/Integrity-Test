What is integrity test?
Integrity test is basically a quiz questionnaire, testing the integrity values in one. Transforming regular Q/A into a game in which one can score points and share to social medias like Facebook & Twitter, Integrity Test aims to create awareness about the moral values & integrity in mass.
Scoring Criteria
  • 2 points will be rewarded for each correct answer
  • 1 point will be deduced for each wrong answer
  • Score will be according to the level
